Este sitio web ha sido diseñado teniendo en cuenta las pautas de accesibilidad para asegurar que la mayor parte de las personas pueda acceder a sus contenidos sin dificultad.

Medidas adoptadas (HTML & CSS)

Las siguientes medidas han sido implementadas en el código para mejorar la accesibilidad y usabilidad del sitio, siguiendo parcialmente las WCAG 2.1:

  • Uso de la estructura semántica de HTML5 (<header>, <nav>, <main>, <footer>, etc.) para definir regiones clave.
  • Uso de encabezados jerárquicos (<h1> a <h6>) para estructurar el contenido de forma coherente.
  • Provisión de texto alternativo (alt) en imágenes relevantes.
  • Tablas de datos marcadas correctamente con <caption>, <th> y el atributo scope.
  • Formularios accesibles con etiquetas explícitas (<label>), agrupaciones (<fieldset>/<legend>) y la indicación de campos obligatorios.
  • Uso de atributos de idioma (lang="es") en las páginas.
  • Implementación de un enlace “saltar a” visible al foco del teclado.
  • Diseño adaptativo (uso de viewport y media queries) para visualizar el contenido correctamente en diferentes tamaños de pantalla.
  • Asegurar un indicador de foco visible en todos los elementos interactivos (:focus).
